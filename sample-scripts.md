# Sample Scripts

Complete, ready-to-drop-in `.lua` files — as opposed to [Snippets](snippets), which are small building
blocks meant to be copied into something you're already writing. Split into three pages, one per
`scripts/` folder, since that's the first thing you need to know about any entry here (see
[Getting Started](getting-started) if `OnBoot`/`OnLoad`/`OnKey` isn't familiar yet):

- **[OnBoot Scripts](sample-scripts-onboot)**
- **[OnLoad Scripts](sample-scripts-onload)**
- **[OnKey Scripts](sample-scripts-onkey)**

Each entry is collapsed by default — title and a one-line description up front, full script behind the
toggle — so these pages stay easy to scan as more get added. Meant to grow with community contributions;
if you've built something reusable, this is where it goes.
