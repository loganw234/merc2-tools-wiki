# About this wiki

This is a community study/reference resource for modding **Mercenaries 2: World in Flames (PC)**, a
long-discontinued game with no current official storefront availability. It documents observed engine
behavior — module structure, functions, events, lifecycle — for people who already own and are running
the game and want to script against it.

Nothing here redistributes game files, assets, or original source code. Pages describe *behavior*
(what a system does, how to call it, what to expect) rather than reproducing the material itself. Where
a page hasn't yet been confirmed against the running game, it's marked with a banner saying so — see
each module reference page.

Not affiliated with Pandemic Studios or Electronic Arts.
