# Reference

Lookup tables — pages you search for one specific fact rather than read start to finish. Distinct from
[Resident Modules](resident/)/[Engine Namespaces](namespaces/)/[lua-bridge API](lua-bridge-api/), which
document *behavior* (what a function does); everything here documents *names* (what string to pass where).

- **[Glossary](glossary)** — terms used throughout this wiki that aren't standard Lua vocabulary or mean
  something specific in this engine.
- **[Hash Lookup](hash-lookup)** — the master name/hash table.
- **[Sound, Music & Effects](sound-music-effects)** — music cue, particle effect, and voice-over key names.
- **[Spawn Reference](spawn-reference/)** — `Pg.Spawn`/`Pg.GetGuidByName` template names, organized by
  task (vehicles, weapons, name lookups).
- **[Model Rig Browser](model-rig-browser)** — an interactive, searchable browser for every bone/hardpoint
  name recovered across all 329 models in the game, with a live 2D skeleton view.
