# Recipes

Two kinds of ready-made content live here, split so you can quickly tell which one you're looking for:

- **[Snippets](snippets)** — small, copy-pasteable building blocks. One value read or written, one
  concept demonstrated. Meant to be pasted into something you're already writing, not run as-is.
- **[Sample Scripts](sample-scripts)** — complete, ready-to-drop-in `.lua` files, organized by which
  `scripts/` folder they belong in (`OnBoot` / `OnLoad` / `OnKey`). Meant to be used as-is.

Both categories are meant to grow with community contributions — snippets and scripts alike. If you've
built something reusable, either page is a good home for it, depending on whether it's a building block
or a finished tool.

Not sure where to start? [Snippets](snippets) first — [Sample Scripts](sample-scripts) makes more sense
once you've seen a few smaller examples and know roughly what a script in this ecosystem looks like.
